# Images folder

Put your photos here. The website reads paths from `src/data/images.js`.

| File                  | Used in              |
| --------------------- | -------------------- |
| `pic-1.jpg`           | Hero (homepage)      |
| `pic-2.jpg`           | About section        |
| `smart-resume-ai.png` | Smart Resume AI card |

**To change a photo:** replace the file in this folder, or edit `src/data/images.js`.

**Recommended:** portrait photos, at least 800×1000 px, JPG or WebP.
